# HSK Chinese Companion

A mobile-friendly Chinese learning companion for HSK-style study, built as a single-page HTML app.

This project is designed to be a smarter alternative to a static PDF phrasebook. It combines searchable vocabulary, common phrases, grammar patterns, audio playback, favourites, and quiz-style practice in one simple browser-based page.

## What it does

- Provides HSK-style Chinese vocabulary and phrases
- Includes English, Chinese characters, and pinyin
- Lets users search by English, pinyin, or Chinese characters
- Plays Mandarin audio using the browser speech engine
- Allows favourites to be saved locally in the browser
- Includes grammar patterns and example sentences
- Includes quiz-style practice
- Works on desktop and mobile browsers
- Can be added to a phone home screen for easier access
